## Glossary of Terms

- **Unit Test:** Test of a single component or function
- **Integration Test:** Test of combined components
- **Refactoring:** Improving code structure without changing external behavior
- **Linting:** Automated checking of source code for style issues
- **CI/CD:** Continuous Integration and Continuous Delivery/Deployment
