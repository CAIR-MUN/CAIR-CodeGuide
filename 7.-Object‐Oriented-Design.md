Object-Oriented Design (OOD) is a programming paradigm centered around
objects that bundle both data and behaviors. Leveraging OOD principles
helps to build modular, maintainable, and scalable systems.

## 7.1 Classes and Interfaces Guidelines

- Keep classes focused (Single Responsibility Principle).
- Prefer composition over deep inheritance trees.

### Example (Python)

```python
class DatabaseClient:
    def __init__(self, connection_pool):
        self._pool = connection_pool

    def fetch(self, sql):
        with self._pool.get() as conn:
            return conn.execute(sql)
```

## 7.2 Encapsulation and Abstraction

- Expose a small, well-documented public API and keep internal helpers
  private.
